🚀 Deployment Guide — 100% Free
1Get Groq API Key (Free LLM)
Go to groq.com → Sign up free → API Keys → Create Key
Copy key → paste as GROQ_API_KEY in your environment variables
Free: 14,400 requests/day
2WhatsApp Cloud API (Free)
Go to developers.facebook.com → Create App → Business → WhatsApp
Copy Access Token → WHATSAPP_TOKEN
Copy Phone Number ID → PHONE_NUMBER_ID
Free: 1,000 conversations/month
3Push to GitHub & Deploy on Render
Upload code to a private GitHub repo → Go to render.com
New Web Service → Connect repo
Build: pip install -r backend/requirements.txt
Start: gunicorn --chdir backend main:app
Add all env vars from .env.example → Deploy → Get your free HTTPS URL
4Set Webhook in Meta Console
WhatsApp → Configuration → Webhook
URL: https://YOUR-APP.onrender.com/webhook
Verify Token: fluw_secret_2024
Subscribe to: messages → Save
5Keep Bot Awake 24/7 (Free)
Render free tier sleeps after 15 min. Use UptimeRobot.com (free)
Monitor type: HTTP → URL: your Render URL → Every 5 min
Your bot will now stay awake 24/7 for ₹0!
🎉 Total Cost: ₹0 Forever
Groq free tier + WhatsApp free tier + Render free tier + UptimeRobot free tier
